Hyper-V快速安装的Ubuntu有时候登录(xrdp)连接没任何反应

发表于 2023/10/13 更新于 2023/11/01
作者 lipeng 1 分钟阅读

原因

因为 xrdp 远程连接时之前你登录的 ubuntu 账号未注销导致

解决方案

点击菜单栏基本会话图标按钮，进入 ubuntu 系统后注销之前登录的账号，再次点击增强会话图标按钮重新登录(xrdp)就行了
